Pre-Release Versions of Event Store

Please note that pre-release versions of the Event Store are not recommended for production use.

When upgrading to v3.0.0 Release Candidate 2 with an existing database, it is necessary to stop all projections which are running in version 2 before upgrading, and restart them when the upgrade is complete.
